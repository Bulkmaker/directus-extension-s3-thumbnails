## [0.4.2] — 2026-07-09
### Fixed
- Запись превью в S3 больше не падает на бэкендах, запрещающих per-object ACL
  (Beget S3 при публичной bucket-политике отбивал `PutObject` с `ACL: public-read`
  как «AccessDenied»). Теперь при отказе по ACL превью пишется без ACL (объект и так
  публичен через bucket-политику), результат кэшируется на процесс.

**Безопасно обновлять:** ✅ да (обратно совместимо: где ACL работает — ставится как прежде).

## [0.4.1] — 2026-07-09
### Changed
- CI-сборка dist; детект устаревших пресетов при смене конфига
- Сборка `dist` — в GitHub Actions (при пуше в `src`), `dist` коммитится в `main`.

**Безопасно обновлять:** ✅ да.
